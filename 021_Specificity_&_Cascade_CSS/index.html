If we use same type of selectors, then the selector written at last will be applied. Inline Style > ID Selector > Class or Attribute Selector > Element Selector > Universal Selector.

We can also use !important. This overwrites even the inline style.

Specificity Calculation To calculate specificity, assign a value to each part of the selector:

  • Universal Selector: 0
  • Element selectors and pseudo-elements: 1
  • Class selectors, attribute selectors, and pseudo-classes: 10
  • ID selectors: 100
  • Inline styles: 1000
!important at the end of a CSS value gets a specificity score of 10,000 points.
