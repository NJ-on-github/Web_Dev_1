rotate

transform: rotate(45deg);
or transform: rotate(0.25turn);
1 is 360 degree, 0.25 is 90 degree, 0/5 turn is 180degree

rotating on axis

transform: rotateX(45deg); transform: rotateY(355deg); transform: rotateZ(15deg);

scale

transform: scale(1.8); transform: scaleX(1.8); transform: scaleY(1.8); transform: scaleZ(1.8); transform: scale3d(,,);

skew

transform: skew(5deg, 6deg); transform: skewX(5deg); transform: skewY(5deg);

There is no skew on z axis.

translate

It is most used property.


transform: translateX(100%); transform: translateX(100px); transform: translateY(100%); transform: translateY(100px);

it has negative values as well. We can use it like transform: translateY(-100%) to hide a nav bar.

Multiple transform properties

transform: translateY(75%) skewX(45deg) translateX(-25%) ;
