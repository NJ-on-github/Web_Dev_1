fr unit 

        instead of
        
        grid-template-columns:100px 50px ;
        we can use
        width: 80vw;
        grid-template-columns:1fr 2fr 1fr ;
        or

Don't use padding in flex box or gris, use gap instead

        gap: 15px 25px;
        or
        row-gap: 15px;
        column-gap: 25px;

We also don't get space in the corners, by using gap.

justify and align

        justify-items: center;
        align-items: center;

stretch is default setting. Don't use flex-end use end instead.
We can also use align-self and justify self for individual items.

repeat()

We can also use repeat function while making the grid. 

            grid-template-columns: repeat(4, 6fr);

minmax

we can set minimum and maximum size of cell as well. 

            grid-template-columns: repeat(4, minmax(100px, 3fr));
This minmax function can be used anywhere.
