keyframes animation-name

            @keyframes frame1 {
                from{}
                to{
                    transform: translateX(500px);
                }
            }
            animation-name: frame1;


animation-duration

How much time should the animation take to complete. 
        animation-duration: 3s;

animation-iteration-count

How much times should the animation repeat. 
        animation-iteration-count: 3s;

animation-timing-function

Properties to time within the animation. 
        animation-timing-function: ease-in-out;

animation-delay

How much time after should the animation start. 
        animation-delay: 1s;

animation-direction

How much time after should the animation start. 
        animation-direction: reverse;
        (Will start animation from end position).
        animation-direction: alternate-reverse;
        (Will start animation from end position and repeat starting from staring positon).
        (Can be used for simple harmonic function)

animation-play-state

It can be running or paused, ideally implemented through javascript. paused state will pause the animation wherever it is (it can be in between any animation). 
        animation-play-state: paused;

Shorthand Property

        animation: name duration timing-function delay iteration-count direction fill-mode;
        animation: frame1 2s ease-in 1s  reverse;
We can miss a property in between as well.

animation-fill-mode

by default - none 
        animation-fill-mode: forwards;
        (the animation will be played until the end and then will be filled with the last value of the animation. (it will stop at end position))
        animation-fill-mode: backwards;
        (will stay at the initial postion(position befire animation) after the animation.)
We can miss a property in between as well.
The keyframes will be applied to the element from the start to the end of the animation. the animation-delay will start after the first keyframe is applied.

keyframes according to percentage

        @keyframes frame2 {
            0% {
                transform: rotate(25deg);
            }
        
            25% {
                transform: rotate(45deg);
            }
        }

Applying multiple keyframes

        animation: frame1 2s ease-in 1s forwards, frame2 2s ease-in 1s 2;
