Container properties

1. We can give display as flex to the container (not items), this makes all the inside element go left to right by default. If the items overflow they start squeezing. Still mostly we use flex as display property. 2. justify-content-center. Centers the items inside the container horizontally. align item center. Centers the items inside the container vertically. 


    justify-content: flex-end;
    /* justify to end (right) of flex box */

    justify-content: flex-start;
    /* justify to left (starting) of flex box */

    justify-content: right;
    /* justify to right of flex box */

    justify-content: space-around;
    /* Space around every element space in between border of container and first and last element as well */

    justify-content: space-between;
    /* Space in between every element and NO space in between border of container and first and last elements */

    justify-content: space-evenly;
    /* puts space evenly between all items and border of the container (same space between every item and border of container).
    whereas
    space around puts half space of space in between elements in between border of container and first and last element */

    
    /* Baseline aligns all the items on theier baseline*/

flex direction

By default left to right. 
    flex-direction: column;
    /* Lays items top to down */
    
    flex-direction: column-reverse;
    /* Lays items down to top */
    
    flex-direction: row;
    /* Lays items left to right */

    flex-direction: row-reverse;
    /* Lays items right to left */

Working of justfiy and align

When flex direction is row. The x-axis(left to right) becomes the main axis and y-axis(top to down) becomes the cross axis (perpendicular to main axis). justify content center means to justify items center on main axis.

if flex direction is row, it centers horizontally.
if flex direction is column, it centers items vertically.
  • main start, cross start,
    • main end, cross end.
  • main size, cross size,

align item * align content

align content is used for multiple rows. align item is used for a single line.
There is no justify items.

flex-flow

flex-flow: <flex-direction> <flex-wrap> 
        ex:
        flex-flow: row wrap;
        /* shorthand property for setting flex direction and flex wrap */

gap

puts specified amount of gap between each item horizontally and vertically.
We can also specify gap differently for rows and columns. gap: 30px; column-gap: 5px; row-gap: 20px;

or a short hand version 30px is row gap 50px is column.
gap: 30px 50px;

Item properties

order

higher the order, more towards the end it shifts.
Default order is 0. Less the order more towards the start it shifts.
If same order number, last written will appear last.

flex-grow

By default 0;
flex grow 1 means every element will take equal space. flex grow 2 means every item will take space of two items.

flex-shrink

Flex shrink 2 means the item will shrink twice as much as ither items when wwrap is off.

align self

To change configuration of a particular item such that it changes its position, it aligns self.
